★ Platform Fees (Trading App)
▼

Transaction fee is a flat amount per swap. Service fee uses whichever is larger: the percentage of the trade value or the minimum — matching common broker rules.

⚡ Network / Gas Fees (Per Trade)
▼

Add one entry per fee type. Each fee is deducted per trade in that token (e.g. 0.0005 BNB gas per swap).

✓ Trade Worthiness Rules
▼

Swaps where fees and slippage eat too much of the notional are flagged. Adjust the rules to match how picky you are.

Friction = total fees + slippage estimate for that row. "Borderline" means approaching your max friction, or the swap is a very thin slice of the portfolio.

≡ Breakeven Price (2 Tokens, One Price Locked)
▼

If you only care about a pair and want to ask: "With token A priced at this PHP level, where should token B trade so friction on the implied rebalance swap is at most my cap?" This uses the same network + platform fee rules as above.
